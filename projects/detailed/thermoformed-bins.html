Custom Thermoformed Storage Bins

Designed and manufactured custom storage solutions for machine shop organization using vacuum thermoforming and 3D printing integration

Machine Shop Project Individual Work Multi-week Project

Project Overview

This project involved designing and manufacturing custom storage solutions to improve organization and efficiency in the machine shop. Using vacuum thermoforming techniques combined with 3D printing for tooling, I created specialized storage bins tailored to specific tools and components.

The project demonstrated advanced manufacturing techniques including vacuum forming, 3D printing for tooling, and product design optimization for manufacturing efficiency and functionality.

Technical Specifications

  • Material: HDPE plastic sheets
  • Manufacturing Process: Vacuum thermoforming
  • Tooling: 3D printed forming dies
  • Design Software: SolidWorks CAD
  • Applications: Machine shop organization and tool storage

Manufacturing Process

  1. Design Phase: CAD modeling of storage requirements and bin geometry
  2. Tooling Creation: 3D printing of forming dies and fixtures
  3. Material Preparation: HDPE sheet cutting and preparation
  4. Vacuum Forming: Heating and forming plastic over custom dies
  5. Finishing: Trimming, edge finishing, and quality inspection

Key Achievements

  • Successfully integrated 3D printing with traditional thermoforming processes
  • Created custom storage solutions optimized for specific tool geometries
  • Improved machine shop organization and workflow efficiency
  • Demonstrated cost-effective small-batch manufacturing techniques
  • Achieved professional surface finish and dimensional accuracy
