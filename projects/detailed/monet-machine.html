Monet Machine - 2.5 DOF Automated Painter

Five-week team project creating an autonomous painting and plotting system with precise brush control

Class Project 5-person team 5 weeks

Project Overview

The Monet Machine was an ambitious five-week group project to create a machine that could automatically paint and draw a given image on a small canvas. This project combined mechanical design, software development, and artistic creativity to bridge the gap between digital art and physical creation.

Our team successfully developed a 2.5 degree-of-freedom robotic system capable of interpreting digital images and recreating them through precise brush movements and paint application. The system demonstrated the integration of image processing, path planning, and mechatronic control.

Technical Specifications

  • Canvas Size: Small format painting surface
  • Degrees of Freedom: 2.5 DOF for X-Y movement and brush control
  • Control System: Arduino microcontrollers with custom programming
  • Software Tools: MATLAB, Inkscape, JSCut for image processing
  • Team Configuration: 5-person engineering team
  • Project Duration: 5-week intensive development cycle

Technical Approach

  1. Image Processing: Inkscape and JSCut used to simplify input images and create optimized toolpaths
  2. Path Generation: Custom MATLAB scripts processed simplified images and generated G-Code instructions
  3. Mechanical System: Designed and built 2.5 DOF robotic platform with precise brush control
  4. Control Programming: Arduino programming for coordinated motor control and painting operations
  5. System Integration: Combined software, hardware, and artistic elements into cohesive painting system

Key Achievements

  • Successfully created automated painting system capable of reproducing digital artwork
  • Developed custom image processing pipeline for optimal painting paths
  • Designed and fabricated precise mechanical positioning system
  • Implemented coordinated control of multiple motors and brush mechanisms
  • Demonstrated successful team collaboration on complex multidisciplinary project

Image Processing
