Monet Painting Machine

Automated painting system that recreates Claude Monet masterpieces using precision robotics, computer vision, and coordinated brush control

Class Project 5-person Team 5 Weeks

Project Overview

Designed and constructed an automated painting system capable of recreating Claude Monet paintings through precision robotics and computer vision. This multidisciplinary project combined mechanical engineering, software development, and artistic analysis to create a system that could analyze famous paintings, translate them into robotic movements, and physically recreate them using brushes and paint. The 5-person team successfully demonstrated the intersection of engineering and art by producing recognizable reproductions of Monet's iconic works through careful coordination of brush positioning, paint mixing, and stroke dynamics.
