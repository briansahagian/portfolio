Monet Machine - 2.5 DOF Automated Painter

Five-week team project creating an autonomous painting and plotting system

Class Project 5-person team 5 weeks

Project Overview

The Monet Machine was an ambitious five-week group project to create a machine that could automatically paint and draw a given image on a small canvas. This project combined mechanical design, software development, and artistic creativity to bridge the gap between digital art and physical creation.

Our team successfully developed a 2.5 degree-of-freedom robotic system capable of interpreting digital images and recreating them through precise brush movements and paint application.

Technical Specifications

  • Canvas Size: Small format
  • DOF: 2.5 degrees of freedom
  • Control: Arduino microcontrollers
  • Software: MATLAB, Inkscape, JSCut
  • Team Size: 5 engineers

Technical Approach

1

Image Processing

We used Inkscape and JSCut to simplify input images and create optimized toolpaths for the painting mechanism.

2

Path Generation

Custom MATLAB scripts processed the simplified images and generated G-Code instructions for precise brush movements.

3

Hardware Control

Arduino microcontrollers interpreted the G-Code and controlled the physical machine's motors and brush mechanisms.

4

Mechanical Design

SolidWorks CAD modeling and 3D printing enabled rapid prototyping and component manufacturing.
