Rod Balancing Transport System

Autonomous vehicle system using Arduino control and motion analysis for stable rod transport

Class Project Individual Project Multi-week

Project Overview

The objective was to design and build a system capable of transporting a 1-foot vertical aluminum bar over a distance of 5 to 10 feet as quickly as possible without the rod toppling over. This challenge required balancing speed optimization with stability control through advanced engineering analysis.

The project demanded a deep understanding of system dynamics, precise motor control, and real-time feedback systems to achieve optimal performance while maintaining the vertical stability of the rod during transport operations.

Technical Specifications

  • Rod Specification: 1-foot vertical aluminum bar
  • Transport Distance: 5-10 feet adjustable range
  • Control System: Arduino microcontroller with custom programming
  • Feedback: Motor encoder systems for real-time monitoring
  • Analysis Tool: SolidWorks Motion Analysis for simulation
  • Optimization: Maximum speed while maintaining stability

Engineering Approach

  1. Motion Analysis: SolidWorks simulation to determine maximum acceleration before rod tipping
  2. Analytical Validation: Mathematical calculations to confirm simulation results and stability thresholds
  3. Control System Design: Arduino programming for optimal velocity curve generation
  4. Real-time Feedback: Motor encoder integration for continuous speed adjustment and error correction
  5. System Integration: Complete mechanical and electrical system assembly and testing

Key Achievements

  • Successfully transported rod without tipping across full distance range
  • Optimized transport speed through comprehensive dynamic analysis
  • Implemented sophisticated control algorithms for smooth acceleration profiles
  • Validated theoretical calculations with practical testing results
  • Demonstrated integration of simulation, programming, and mechanical design

Engineering Analysis

Dynamic Stability Analysis

The critical aspect of this project was determining the maximum acceleration that could be applied without causing the rod to tip over. Using principles of dynamics and moment analysis, I calculated the theoretical maximum acceleration based on the rod's center of gravity and the system's geometry.

SolidWorks Motion Validation

The simulation confirmed my analytical calculations, providing visual verification of the rod's behavior under different acceleration profiles. This dual approach ensured both theoretical understanding and practical validation of the design parameters.

Control Algorithm Development

The Arduino control system was programmed to generate smooth acceleration and deceleration curves that stayed within the calculated safety margins while optimizing transport time. The algorithm incorporated real-time feedback to maintain precise speed control throughout the journey.
